# vue-vant-webApp

**在线查看地址(移动端):** http://yaobawu.cn1.utools.club/dist/#/

> 移动端webApp vue框架vant库 & axios & vue-router & rem百分比布局
> 感谢穆仙念线提供接口

**接口地址** https://www.mxnzp.com/

## 项目运行

```bash
# 克隆
git clone https://github.com/mh185/vue-vant-webApp.git

# 进入项目文件夹
cd vue-vant-webApp

# 安装依赖
npm install

# 运行
npm run dev
```

使用浏览器打开并按F12切换到移动端 http://localhost:9528

## 项目打包

```bash
npm run build:prod
```
